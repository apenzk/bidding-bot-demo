# Bidding Bot Demo

Learn how to post a collection offer on OpenSea!

## Local development

### Setup node (optional)

```sh
nvm install
```

### Install dependencies

```sh
yarn
```

### Create a Wallet

Follow the steps from the command below to create a new wallet and add the private key to a `.env` file in this directory.

```sh
yarn create-wallet
```

### Make an Offer

```sh
yarn bidding-bot
```
